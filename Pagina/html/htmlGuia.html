Introducción

HTML o Hyper Text Markup Language es un lenguaje para describir el contenido de páginas web. Las unidades fundamentales del lenguaje son los elementos html identificados por "tags" : < >. Éstos tienen atributos que los identifican o modifican su contenido.

Tags básicos
Tag Descripción
<html> Define la raiz de un documento HTML
<body> Define el cuerpo
<head> Contiene información que no se muestra en la página, como estilo y metadata
<hn> Define un encabezado n=1...6
<hr> Establece un cambio de tema en la página
<p> Define un nuevo párrafo
<br> Da un salto de línea
<a> Define un link
<img> Define una imágen
<pre> Texto preformateado
<q> Citas
<blockquote> Cita más extensa
<abr> Abreviaciones
<cite> Título de un trabajo
<map> Define un conjunto de divisiones de una imágen
<area> Define cada una de las divisiones para el tag anterior
<div> Define un bloque que engloba elementos html
<span> Define un contenedor de texto

Ejemplos:

<h1>

<h2>

<h3>

<h4> ... etc

CSS

CSS es un lenguaje que sirve para definir el diseño de una página HTML. Puede utilizarse por medio del atributo style en un elemento HTML, dentro del tag <style>, o en un archivo externo referenciado con el tag <link> (ambos tags deben estar dentro de <heading>)

Propiedades CSS básicas
Nombre Descripción Ejemplo
color Color del texto de un elemento html color:blue
font-family Tipo de letra de un texto font-family: verdana
font-size Tamaño de la letra font-size: 160%
border Define el borde de un elemento border: 1px solid powderblue
padding Define el espacio entre el texto y el borde de un elemento padding: 30px
margin Define espacio fuera del borde de un elemento margin: 50px
background-color Color del fondo de un elemento background-color: powderblue
float Define la posición de una imágen con respecto a un bloque de texto float:right
text-align Define la posición del texto text-align: left
border-collapse Junta los bordes de una tabla y sus celdas border-collapse: collapse
border-spacing Espaciamiento de un borde en una tabla border-spacing: 5px

Tablas y listas

Dos elementos html utiles para organizar información son las tablas y listas

Tags para definir una tabla
<Table> Define una tabla
<tr> El renglón de una tabla
<td> La celda de una tabla
<caption> Define el título de una tabla

Atributos de elementos de tablas
colspan Hace que una celda de una tabla ocupe más de una columna
rowspan Expande una celda de una tabla a más de un renglón

Existen dos tipos de listas: listas ordenadas que marcan cada uno de sus elementos con números, listas no ordenadas que marcan cada uno de sus elementos indistintamente.

Tags para definir listas ordenadas y no ordenadas
<ol> Define una lista ordenada
<ul> Define una lista no ordenada
<li> Define un elemento de la lista
<dl> Define una lista de descripción
<dt> Define el término de una descripción
<dd> Define la descripción

Algunos atributos importantes de las listas

list-style-type
Propiedad css para definir los marcadores de los items
type
Atributo html para tablas ordenadas que define el tipo de marcador
float:left y display:inline
Propiedades css para hacer una tabla horizontal

Head y metadata

El tag <head> sirve para incluir información que no se mostrará en la página web, pero que es util para diferentes programas al manejar el archivo HTML. Dentro de éste se utiliza el tag <metadata> que tiene diferentes atributos utiles, entre ellos:


Atributos de metadata
Define el conjunto de caractéres charset = "UTF-8"
Da una descripción de la página web name ="description" content ="Free web tutorials"
Define keywords para un buscador name ="keywords" content ="HTML, CSS"
Define el autor name ="author" content ="Jogn Doe"
Define un lapso en que se debe recargar la página en segundos http-equiv="refresh" content ="30"
Utiliza el viewport para escalar la página al tamaño del aparato que la visualiza name ="viewport" content ="width=device-width, initial-scale=1.0"

Layout

La dispoción del texto en una página se puede determinar con CSS, pero aun así existen tags que ayudan a organizzar los elementos de una página

Tags para definir elementos del layout
Tag Descripción
<header> Encabezado
<nav> Menú de navegación
<section> sección
<article> Artículo independiente
<aside> Contenído lateral
<footer> Contenido en el pie de la página
<details> Información extra
<summary> Sirve de encabezado para <details>

Forms

El elemento <form> sirve para que un usuario puede ingresar y mandar información. Utiliza el atributo action para definir la página a donde se mandará la información.

Elementos notables de form
Elemento Descripción
<input> Elemento para introducir información
<textarea> Cuadro de texto
<label> Texto relacionado a un input
<fieldset> Agrupa elementos de un form
<select> Drop down list
<optgroup> Agrupa elementos de un drop down list
<option> Define una de las opciones del drop down list
<button> Botón

Por su importancia, el elemento input tiene varios atributos importantes

Atributos importantes de input
Atributo Descripción
value Valor inicial
readonly Hace que el valor no se pueda modificar
disabled Desabilita el input
size tamaño
maxlength Tamaño máximo de la entrada

Valores notables del atributo type
Atributo Descripción
text Linea de texto
password contraseña
submit botón que manda la información
reset restablece los datos ingresados en el form a los datos default
radio botón de selección excluyente
checkbox botón de selección incluyente
