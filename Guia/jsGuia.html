Objetos en JavaScript

En JS un objeto es un arreglo de parejas key : value, puede guardar valores primitivos y funciones.
Ejemplo:
var person = { firstName: "John", lastName : "Doe", id : 5566, fullName : function() { return this.firstName + " " + this.lastName; }};
Los atributos se pueden accesar como :

  • objeto.nomAtributo
  • objeto["nomAtributo"]
  • objeto.nomFuncion()
  • objeto.nomFuncion -Da la definicion de la funcion-

Algunos metodos de string

En js los string NO se pueden accesar como arreglos ("str[indice]"), en los metodos se empieza a contar caracteres por el 0.

  • str.length
  • str.indexOf()
  • str.lastIndexOf()
  • str.search()
  • str.slice()
  • str.substring()
  • str.substr()
  • str.replace()
  • str.toUpperCase()
  • str.charAt()
  • str.split("separador")

Arreglos

Estos son algunos metodos nativos de los arreglos en JS

  • arr.push(nuevo elem) - añade al final
  • Array.isArray(posibleArreglo)
  • arr.toString() - da los elementos separados por comas
  • arr.join("separador")
  • arr.pop() - regresa y saca el ultimo elemento
  • arr.shift() - recresa y saca el primer elemento y recorre los demas
  • arr.unshift(nuevo elem) - inserta al inicio y regresa el nuevo tamaño
  • arr.splice(pos, num elementos a quitar, elem insertar,elem insertar...)
  • arr.concat()
  • arr.slice(pos) - regresa nuevo arreglo empezando en pos
  • arr.sort() - por orden alfabetico
  • arr.reverse()
  • arr.sort(function(a,b){return a-b}) - ordena números
  • arr.sort(function(a, b){return 0.5 - Math.random()}) - da orden aleatorio
