Introducción

Css es un lenguaje que determina el diseño en el que se desplegara una página HTML. Su importancia radica en que separa el contenido textual de una página de su diseño gráfico.
Su sintaxis básica es:

Selector Declaración
h1 {color:blue;}

El selector puede ser por elemento, por id o por clase:

Por elemento p {color:blue}
Por id #idArbitrario {color:blue}
Por clase .nombreclase {color:blue}

Además, el código css se puede escribir en un elemento por medio del atributo style, dentro del html en el elemento <head> o en una página .css referenciada en el elemento
<link rel="stylesheet" type="text/css" href="archivo.css" >

Colores y background

Los colores se pueden especificar de tres formas

Color rojo Código
background-color:rgb(255,0,0) Valor rgb
background-color:#FF0000 Valor hexadecimal
background-color:red Nombre

La propiedad background tiene las siguientes formas

  • background (color - image - repeat - attachment - position)
  • background-attachment
  • background-color
  • background-image
  • background-position
  • background-repeat

Border , Margin y Padding

La propiedad border define el borde de un elemento

  • border (width - style - color)
  • border-(top;right;bottom;left)
  • border-color
  • border-style
  • border-radius

Cada border-posición se le puede añadir los otros sufijos, por ejemplo:
border-top-color
Además se puede escribir de forma compacta la definicion de cualquiera de los borders:
border-top:width style color

Este es un ejemplo
style="width:300px;border: 4px dotted blue"

La propiedad margin define la distancia entre un elemento y el elemento que lo contiene, sus formas son:

  • margin: top right bottom left
  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

Este es un ejemplo de un párrafo con el márgen modificado
style="background-color:lightblue;margin: 20px 40px 30px 300px"


La propiedad padding determina la distancia entre el contenido de un elemento y su borde, sus formas son :

  • padding: top right bottom left
  • padding-top
  • padding-right
  • padding-bottom
  • padding-left

Este es un ejemplo de un párrafo con el padding modificado
style="background-color:lightblue;padding: 40px 20px"

Propiedad position

La propiedad posición define con respecto a qué se va a determinar la posición de un elemento

Position
Valor Descripción
static El elemento se posiciona automaticamente con respecto a la página, las propiedades top, bottom, left, and right no lo afectan
relative La posición se define con respecto a la posición static del elemento
fixed La posición se define con respecto a la ventana del browser (viewport)
absolute La posición se define con respecto a un elemento ancentro con propiedad posición diferente de static

Otro atributo importante es el z-index, que determina el orden en que se sobreponen los elementos

Overflow y float

Overflow es una propiedad que determina qué hacer con texto cuando se sale del elemento que lo debería contener. La propiedad se puede especificar para ser horizontal o vertical:

  • overflow
  • overflow-x
  • overflow-y

Algunos valores que toma:

  • visible - Se muestra el texto que se desborda
  • hidden - se corta el texto que se desborda
  • scroll - añade una barra para visualizar el texto que se desborda
  • auto - añade barra si es necesario

Combinators, Pseudo-class y Pseudo-element

Son elementos para usar con los selectores :

Combinator Descripción
(espacio) Selecciona a todos los elementos anidados
> Selecciona los elementos anidados solo un nivel debajo
+ Selecciona el siguiente hermano
~ Selecciona todos los hermanos

Pseudo-class se utiliza con el selector para seleccionar tipos predefinidos:

  • :hover - cuando un elemento tiene el cursor encima
  • :first-child - selecciona si es primer hijo
  • :focus - Seleccina los inputs a los que se les ha dado click

Pseudo-elements se utiliza para seleccionar partes predefinidas

  • ::after
  • ::before
  • ::first-letter
  • ::first-line
  • ::selection
